# Music Tools

**Music Tools** is a modern, interactive suite of utilities designed for musicians, songwriters, and students. Built with a focus on minimalism and usability, this application provides real-time visualization for scales, chords, and tuning across multiple instruments including Piano, Guitar, and Bass.

## Features

*   **Interactive Fretboard & Keyboard:** Visualize scales and chords dynamically on guitar, bass, and piano.
*   **Scale Library:** Comprehensive database of musical scales including Major, Minor, Pentatonic, Blues, and Modes.
*   **Chord Identification:** Select notes on an instrument to instantly identify the chord name and root.
*   **Smart Tuner:** Chromatic tuner with visual feedback for precise instrument tuning.
*   **Metronome:** Accurate timing tool with customizable BPM and time signatures.
*   **Responsive Design:** Fully optimized for both desktop studio usage and mobile on-the-go practice.

## Tech Stack

*   **Frontend:** React 18, TypeScript
*   **Styling:** Tailwind CSS
*   **Audio:** Web Audio API (Oscillators, AnalyserNodes)
*   **Analytics:** Vercel Analytics
*   **Deployment:** Vercel

## Data Source

The musical data structures and logic used in this application are inspired by and compatible with the [RiffTheoryBackend](https://github.com/ShirelleW/RiffTheoryBackend) database structure.
